# API.JSON Generator

Утилита для создания файла `api.json` с хеш-суммами и размерами файлов. Предназначена для использования в лаунчерах игр или системах проверки целостности файлов.

##  Функциональность

- Рекурсивное сканирование директории `files`
- Вычисление SHA-256 хешей для каждого файла
- Измерение размера файлов
- Сохранение данных в формате JSON
- Красивый прогресс-бар с использованием библиотеки `rich`
- Обработка ошибок и прерываний
